# WebLLM Service

Integration layer for Web-LLM library, providing LLM inference capabilities in the browser. Manages LLM model initialization, downloads, and response generation with quantized model support.

## Responsibilities

- Initialize Web-LLM models with proper configuration
- Generate responses using LLM models
- Manage available quantized model options
- Download and cache LLM models
- Handle model initialization with caching
- Track current model and loading state
