# Transformers Service

Integration layer for Transformers.js library, providing text generation and model management capabilities. Handles model initialization, downloads, and response generation using transformer-based models.

## Responsibilities

- Initialize Transformers.js models
- Generate text responses using transformer models
- Manage available models list and descriptions
- Download model weights from HuggingFace
- Handle model initialization caching and promises
- Track current model and loading state
